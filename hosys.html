Hosystem

Hosystem was a class project that involves creating a hospital management system designed for taking appointment bookings and assigning them to relevant doctors. Doctors may also register an account and manage appointment statuses with their patients. More hospitals and sectors can be added into the system as well.

The system was built using PHP, HTML, and Tailwind CSS. The database runs on SQL.

From this project, I've learned how to integrate newly created values (doctor accounts/hospitals/patients etc.) into options for both patients and doctors to select and other web design aspects that caters to two types of accounts running on a system. Further improvements can be made to the system in the future to better user experience.
