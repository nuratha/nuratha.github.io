Schedual

Schedual was created as my project where it takes a spin on regular scheduling/time management apps by integrating game mechanics inspired by classic RPG-type games such as earning task completion points, late penalties, health, weekly ranking and leveling.

There is also a section to "converse" with an avatar using an option-based dialogue style that issues productivity-related advice based on your situation. The idea was to create a more interactive and fun approach to productivity.

The system was built using PHP, HTML, and Tailwind CSS, alongside ChartJS for the visual productivity charting. The database runs on SQL.

Throughout this project, I've learned several aspects of web design and additional elements that could be implemented for a more engaging experience. More improvements can be done to this project over time to create a better system.
